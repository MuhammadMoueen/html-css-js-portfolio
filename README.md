# HTML, CSS & JavaScript Portfolio

This workspace contains a collection of practice exercises and real projects built with plain HTML, CSS, and JavaScript. It demonstrates core front-end skills, DOM manipulation, event handling, API integration, and interactive UI features.

## Why this repo is strong for GitHub

- Includes both practice exercises and real applications
- Uses vanilla JavaScript without frameworks, showing raw front-end skills
- Demonstrates API usage, state handling, and UI feedback
- Contains small, polished projects ideal for showcasing growth as a web developer
- Easy to run locally by opening each project HTML file in the browser

## Workspace structure

- `Practice Work/` — beginner and practice examples
- `Real Projects/` — more complete applications and interactive demos

## Practice Work projects

### Button-Like
- Simple like counter built with a button and text
- Demonstrates event handling and DOM updates in `button-like.js`

### Home / Learning page
- Example practice page for HTML layout, CSS styling, and JS script linking
- Shows early experimentation with HTML structure and JS behavior

### Basic DOM practice
- `first.js` contains core JavaScript practice covering:
  - variables and data types
  - conditionals and loops
  - arrays and objects
  - string manipulation
  - basic DOM and event examples

## Real Projects

### Currency & Crypto Converter
- Located in `Real Projects/Currency_Converter`
- Converts fiat currencies and cryptocurrencies
- Uses the Frankfurter API for fiat exchange rates
- Uses the CoinGecko API for crypto pricing
- Loads currency flags and crypto icons dynamically
- Provides fallback fiat rates when the API fails
- Built with modular JavaScript files: `api.js`, `crypto.js`, `codes.js`, `ui.js`, and `script.js`

### Dark Color Mode Toggle
- Located in `Real Projects/Dark_Color`
- Theme toggle button switches between light and dark mode
- Uses JavaScript to toggle CSS class and update button text
- Demonstrates UI state toggling and accessible interaction

### Mouse Hover Effect
- Located in `Real Projects/Mouse_Hover`
- Interactive card that changes style and text on hover
- Good example of combining CSS transitions with JS events

### Rock Paper Scissors Game
- Located in `Real Projects/Rock_Paper_Scissors`
- Browser game with 10 rounds, score tracking, and final result
- Includes choice buttons, animations, and restart behavior
- Demonstrates game logic, DOM updates, and UI feedback

## How to use this repo

1. Open the folder in VS Code
2. Open any project folder
3. Launch the corresponding HTML file in your browser

For example:
- `Practice Work/button-like.html`
- `Real Projects/Currency_Converter/index.html`
- `Real Projects/Dark_Color/dark-color.html`
- `Real Projects/Mouse_Hover/mouse-hover.html`
- `Real Projects/Rock_Paper_Scissors/index.html`
